## ğŸ“… Annual Checklist

### Annual Inspection (12 Calendar Months)
- [ ] **Due**: Every 12 calendar months (can use previous month if done in first 15 days)
- [ ] Schedule 30-60 days in advance
- [ ] Budget: $1,200-$2,500 (single-engine piston)
- [ ] Prepare aircraft logbooks for mechanic
- [ ] Review all open squawks with mechanic
- [ ] Provide list of pilot-reported issues
- [ ] Confirm all ADs will be reviewed
- [ ] Plan for aircraft downtime (2-3 weeks typical)
- [ ] After inspection: Review completed work
- [ ] After inspection: Verify return-to-service entry
- [ ] After inspection: File paperwork in permanent records

### ELT Inspection (12 Calendar Months)
- [ ] **Due**: Every 12 calendar months
- [ ] Often combined with annual inspection
- [ ] Check battery expiration date (50% life or 1 cumulative hour use)
- [ ] Replace battery if needed ($150-$400)
- [ ] Verify 406 MHz ELT registered with NOAA
- [ ] Update registration if ownership changed
- [ ] Verify signal strength adequate

### Registration Renewal (7 Years)
- [ ] **Due**: Every 7 years from issue date
- [ ] Set reminder 6 months before expiration
- [ ] Renew online at faa.gov
- [ ] Fee: $5
- [ ] Processing time: 4-6 weeks
- [ ] Update aircraft records when received

### Customs Decal (Annual - International Only)
- [ ] **Due**: Calendar year (January 1 - December 31)
- [ ] Purchase through DTOPS: dtops.cbp.dhs.gov
- [ ] Cost: $31 (2025 rate)
- [ ] Affix decal to aircraft in visible location
- [ ] Retain proof of purchase

### Insurance Renewal (Annual or As Required)
- [ ] Review coverage 90 days before expiration
- [ ] Shop for competitive quotes
- [ ] Verify pilot requirements match current operations
- [ ] Update hull value if needed
- [ ] Confirm international coverage if applicable
- [ ] File copy of new policy in aircraft records

- [ ] Required for IFR flight in controlled airspace
- [ ] Schedule with certified repair station
- [ ] Cost: $200-$300 (standalone) or bundled with transponder
- [ ] Tests per Part 43 Appendix E
- [ ] Static system leak test
- [ ] Altimeter accuracy at multiple pressure altitudes
- [ ] Bundle with transponder inspection

- [ ] **Due**: Every 24 calendar months (combined with 91.411)
- [ ] Pitot tube and drain inspection
- [ ] Static port inspection
- [ ] Alternate static source test (if equipped)
- [ ] Instrument accuracy checks

## â±ï¸ Hour-Based Checklist

### 100-Hour Inspection (For Hire Operations)
- [ ] **Due**: Every 100 hours tachometer time
- [ ] **Applicability**: Aircraft used for compensation or hire
- [ ] Can exceed by 10 hours to reach inspection facility
- [ ] Excess time deducted from next 100-hour interval
- [ ] Track tachometer time (not Hobbs)
- [ ] Ground aircraft 10 hours before due (scheduling buffer)
- [ ] Annual inspection satisfies 100-hour requirement
- [ ] Cost: $1,200-$2,500 (similar to annual)

### Oil Changes
- [ ] **Due**: Per manufacturer (typically 25-50 hours)
- [ ] Review oil analysis trends (Part 135)
- [ ] Check oil filter for metal particles
- [ ] Document in maintenance logbook

### Engine TBO Compliance (Part 135 Only)
- [ ] **Required**: Part 135 must comply with manufacturer TBO
- [ ] Track time since major overhaul
- [ ] Plan for engine overhaul 100+ hours in advance
- [ ] Budget: $20,000-$40,000+ (piston engine)
- [ ] Ground aircraft when TBO reached

- [ ] Prepare customs documentation
- [ ] File international flight plan
- [ ] Verify destination country requirements

## ğŸ¯ Compliance Tips

### Best Practices
1. **Set Multiple Reminders**: 60 days, 30 days, 15 days before due
2. **Track Tach Time**: Use tachometer (not Hobbs) for 100-hour compliance
3. **Bundle Inspections**: Combine 24-month inspections to save money
4. **Schedule Early**: Book maintenance 30+ days in advance
5. **Budget Ahead**: Set aside 10-15% of aircraft value annually for maintenance
6. **Review Monthly**: Check all compliance dates first of each month
7. **Document Everything**: If it's not in the logbook, it didn't happen
8. **Know Your Operation**: Understand Part 91/61/141/135 requirements

### Common Mistakes to Avoid
- âŒ Using Hobbs meter for 100-hour calculations (use tach!)
- âŒ Forgetting 7-year registration renewal
- âŒ Not budgeting for annual inspection discrepancies
- âŒ Flying with expired VOR check when using VOR for IFR
- âŒ International flight without proper FCC license
- âŒ Operating Part 135 without manufacturer TBO compliance
- âŒ Incomplete logbook entries (missing signatures/certificate numbers)
